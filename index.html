Methodology

We employ four complementary empirical strategies to establish causality and construct validity, following best practices from Petersen (2009), Harvey et al. (2016), and Kolari & Pynnönen (2010).

Data & Sample Construction

Sample Period & Coverage

509 bank 10-K and 10-Q filings from 50 SEC-registered banks (2023-2025)

CNOI Construction

Composite opacity index with 7 dimensions:

$$\text{CNOI} = 0.2D + 0.2G + 0.2R + 0.1J + 0.1T + 0.1S + 0.1X$$
  • D (Definiteness): Specificity vs. vague ranges
  • G (Granularity): Detail level in loan segmentation
  • R (Reconcilability): Quarter-to-quarter tracking
  • J (Justification): Methodology disclosure
  • T (Timeliness): Lag in disclosure updates
  • S (Stability): Consistency across periods
  • X (Cross-reference): Internal linkage quality

Market Data

Yahoo Finance (daily prices) + Ken French Data Library (factor returns)

Empirical Methods

1. Decile Portfolio Analysis

Sort banks into 10 groups by CNOI, track forward returns with multiple lag structures

$$r_{LS} = r_{D1} - r_{D10}$$

2. Event Study (SVB Crisis)

March 2023 Silicon Valley Bank crisis as natural experiment. Robust tests:

  • BMP (Boehmer-Musumeci-Poulsen)
  • Corrado Rank Test
  • Sign Test
  • Kolari-Pynnönen (cross-sectional correlation adjustment)

3. Panel Regression

Fixed effects model with Driscoll-Kraay standard errors:

$$r_{i,t+1} = \alpha + \beta \cdot \text{CNOI}_{i,t} + \gamma X_{i,t} + \mu_i + \lambda_t + \varepsilon_{i,t}$$

4. Construct Validation

Horse-race regressions vs. Fog Index, incremental R² tests, ICC reliability

Robustness & Validity

Multiple Testing Corrections

Deflated Sharpe Ratio (DSR) - Bailey & López de Prado (2014)

$$\text{DSR} = \frac{\text{SR}}{\text{SR}_{\text{threshold}}}$$

Adjusts for testing 12 CNOI variants with Šidák correction

Superior Predictive Ability (SPA) - Hansen (2005)

Guards against data snooping across entire specification universe

Few-Cluster Inference

Wild Cluster Bootstrap - Cameron, Gelbach & Miller (2008)

999 replications with Rademacher, Mammen, and Normal weights for robust p-values when clusters <20

Event Study Corrections

Kolari-Pynnönen Adjustment - Kolari & Pynnönen (2010)

$$t_{\text{KP}} = \frac{t_{\text{standard}}}{\sqrt{1 + (N-1)\bar{\rho}}}$$

Corrects for cross-sectional correlation when events cluster in calendar time (SVB crisis)

Data Integrity

  • SHA-256 checksums on factor data (prevents phantom alphas)
  • Delisting return adjustments (-30% Shumway 1997)
  • CRSP format tracking (FIZ pre-2025, CIZ post-2025)
  • NYSE breakpoint conformity
Statistical Inference Policy

Standard Errors & Clustering

  • Time-series: Newey-West HAC (6 lags)
  • Panel: Two-way clustering (firm × time)
  • Few clusters (<20): Wild cluster bootstrap (mandatory)
  • Cross-sectional: Heteroskedasticity-robust (HC3)

Multiple Testing

  • DSR: Šidák correction for n_trials = 12
  • Harvey-Liu-Zhu: Target |t| > 3.0
  • Bonferroni: When testing K hypotheses simultaneously
